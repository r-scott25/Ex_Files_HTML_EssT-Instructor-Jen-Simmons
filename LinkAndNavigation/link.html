Links

The Web is called a web because of how different parts link link to other parts, creating a web.

To Make a Link, use the <a> element.

a stands for Anchor

In the opening tag, use the href attribute. The href points to where we want the link to go.
href stands for Hypertext Reference.

<a href="page.html>Link</a>

Example Link Text

The a element is an inline element. We can wrap links around anying, not just text.

Example: We can wrap text around an image element:

You can wrap link tags around more complex content, like a teaser card.
You can the a element around several other elements, creating a whole group of things that all become linked.

What Can Go in A Link Element?

If you are linking to something out on the web, like some other website,
you can create the link by putting the whole URL in the slot.
It doesn't actually matter whether you include the trailing slash or not.

Example:

Link
<a href="https://example.com>"Link</a>

Some URLs point to the homepage of a site. Others point to content deeper in.
All of these are called an absolute URL. You point to a specific place on the web, an absolute place.
You must include the HTTP or the HTTPS in an absolute URL.

HTTP

Hypertext Transport Protocol

This is the way that everything on the web talks to everything else. The protocol, the rules of the communication system.

HTTPs

Hypertext Transport Protocol Secure

In the beginning, all websites used HTTP. Today, experts agree that every website should be using HTTPS.
